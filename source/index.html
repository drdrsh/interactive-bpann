Interactive demo for Artificial neural networks

This is a very minimal demo that aims to serve as a visualization tool for instructors to help their students better grasp artificial neural networks. This demo allows a step-by-step iteration through a fully-connected backpropagation artificial neural network network to get some intuition as what goes on during the process of training. It uses the latest Javascript technologies, particularly the new keyword "yield" that allows clean loop pause and resume.

The code is available on github and is released under MIT license. If you wish to contact me send me an email to mostafa.mohmmed@gmail.com.
